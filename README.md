Privacy Policy
Last updated: June 2026

KR Card Image Export Helper does not collect, store, transmit, sell, or share any personal information.

Data Processing
The extension reads card image URLs and card information from the Korean ONE PIECE Card Game card search page only when the user explicitly activates the extension.

This information is processed locally within the user's browser for the sole purpose of generating PNG card image files.

Data Collection
The extension does not collect:

Names
Email addresses
Phone numbers
Authentication credentials
Financial information
Location information
Browsing history
Analytics data
Data Sharing
No user data is transmitted to external servers.

No user data is sold, shared, or transferred to third parties.

Storage
Generated image files are saved only to locations selected by the user through Chrome's download functionality.

The extension does not maintain any remote database or cloud storage.

Third-Party Services
The extension does not use third-party analytics, advertising, tracking, or remote code services.
